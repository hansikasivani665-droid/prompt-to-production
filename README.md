# Case 1 – Complaint Classifier (AI Code Sarathi)

This project classifies municipal complaints into categories and identifies high-priority issues to fix "severity blindness."

## Features
- **ML Classification**: Categorizes complaints into Infrastructure, Utilities, etc.
- **Priority Detection**: Automatically flags dangerous issues (e.g., potholes, injuries) as **HIGH PRIORITY**.

## How to run

1. **Install Python 3.11+**
2. **Install dependencies**:
   ```bash
   pip install pandas scikit-learn
   ```
3. **Run the classifier**:
   ```bash
   python classifier.py
   ```
4. **Usage**: Enter complaint text when prompted. Type 'exit' to quit.
